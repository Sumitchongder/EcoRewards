# Code of Conduct

## Our Commitment

EcoRewards is committed to fostering a respectful, inclusive, and
collaborative environment for everyone involved in the project.
We welcome contributors, users, and collaborators from all backgrounds
who share a common goal of promoting sustainability and responsible reuse.

---

## Expected Behavior

Participants are expected to:

- Be respectful and considerate in all interactions
- Provide constructive feedback and technical discussions
- Respect differing opinions and approaches
- Maintain professionalism in issues, pull requests, and discussions
- Support a welcoming environment for newcomers

---

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, discrimination, or offensive language
- Personal attacks or intimidation
- Publishing private or sensitive information without consent
- Deliberate attempts to disrupt the project or community
- Any behavior that undermines the projectâ€™s sustainability mission

---

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues and pull requests
- Project documentation
- Community discussions and communications

---

## Enforcement

Project maintainers reserve the right to:

- Remove content that violates this Code of Conduct
- Temporarily or permanently restrict participation
- Take appropriate corrective action when necessary

---

## Reporting

If you experience or witness unacceptable behavior, please report it
privately to the project maintainer.

All reports will be handled confidentially and fairly.

---

## Attribution

This Code of Conduct is inspired by the Contributor Covenant
